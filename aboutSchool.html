School System

A few things more...

Info

This is a school administrative system that I build using java and JDBC as a project for one of the classes at my university. I also used Scene Builder to build the front-end and JDBC using MySQL as a database. The main purpose of this app is to let all teachers who have an account in the system to insert new students as well as to check out all of their current students and their information.

Build With: Java JFrame JDBC & MySQL
